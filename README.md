# CM0 Desk Guardian

## 项目简介
这是一个基于 CM0 Dev Kit 和 ESP32 的桌面智能管家系统。

系统目标：
- 检测主人是否在位
- 统计专注时间
- 提供守护模式
- 检测异常靠近并触发提醒
- 通过 OLED、蜂鸣器、LED 灯环进行状态反馈
- 后续提供本地网页状态面板

## 硬件清单
- CM0 Dev Kit
- ESP32 + 扩展板
- Logitech Brio 100 摄像头
- 4位独立按键模块
- OLED 屏
- 有源蜂鸣器
- LED 灯环
- 杜邦线

### Week 1

### Week 3
